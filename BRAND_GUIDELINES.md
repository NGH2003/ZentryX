## Typography

### Font Family
**Inter** (Google Fonts)
- Professional, modern, highly legible
- Excellent for UI and body text
- Variable font with multiple weights

### Font Weights
- **Light:** 300 (rarely used)
- **Regular:** 400 (body text)
- **Medium:** 500 (emphasis)
- **Semi-Bold:** 600 (subheadings)
- **Bold:** 700 (headings)
- **Extra-Bold:** 800 (hero text)

### Type Scale

#### Display / Hero
- **Size:** 56-72px
- **Weight:** Bold (700) or Extra-Bold (800)
- **Line Height:** 1.1
- **Usage:** Landing page heroes, major announcements

#### H1
- **Size:** 42px
- **Weight:** Semi-Bold (600)
- **Line Height:** 1.2
- **Usage:** Page titles

#### H2
- **Size:** 30px
- **Weight:** Bold (700)
- **Line Height:** 1.3
- **Usage:** Section headings

#### H3
- **Size:** 22px
- **Weight:** Semi-Bold (600)
- **Line Height:** 1.4
- **Usage:** Subsection headings

#### Body
- **Size:** 16px
- **Weight:** Regular (400)
- **Line Height:** 1.6
- **Usage:** Main content

#### Small / Caption
- **Size:** 13px
- **Weight:** Medium (500)
- **Line Height:** 1.5
- **Usage:** Labels, captions, metadata

### Typography Best Practices
- ‚úÖ Use sentence case for most text
- ‚úÖ Use title case for headings
- ‚úÖ Maintain consistent line heights
- ‚úÖ Limit line length to 60-80 characters
- ‚úÖ Use proper hierarchy (don't skip levels)

## Spacing System

### 8-Point Grid
All spacing should be multiples of 8px:

- **4px** - Tiny gaps (icon spacing)
- **8px** - Small gaps (tight spacing)
- **12px** - Compact spacing
- **16px** - Default spacing
- **24px** - Medium spacing
- **32px** - Large spacing
- **48px** - Extra large spacing
- **64px** - Section spacing

### Component Spacing
- **Buttons:** 12px vertical, 24px horizontal padding
- **Cards:** 24px padding
- **Sections:** 64-80px vertical padding
- **Grid Gaps:** 24px (mobile), 32px (desktop)

## Animation Guidelines

### Timing
- **Fast:** 200ms (hover states)
- **Medium:** 300ms (transitions)
- **Slow:** 500ms (page transitions)

### Easing
- **Standard:** `ease-out` (most animations)
- **Bounce:** `cubic-bezier(0.68, -0.55, 0.265, 1.55)` (playful)
- **Smooth:** `cubic-bezier(0.4, 0.0, 0.2, 1)` (material design)

### Animation Types
- **Fade In:** Opacity 0 ‚Üí 1
- **Slide Up:** TranslateY(20px) ‚Üí 0
- **Scale In:** Scale(0.95) ‚Üí 1
- **Lift:** TranslateY(0) ‚Üí -8px
- **Glow:** Shadow intensity increase

### Best Practices
- ‚úÖ Use GPU-accelerated properties (transform, opacity)
- ‚úÖ Reduce motion for accessibility
- ‚úÖ Keep animations subtle
- ‚úÖ Don't animate layout properties (width, height)

## Accessibility

### Color Contrast
- **Normal Text:** Minimum 4.5:1 ratio
- **Large Text:** Minimum 3:1 ratio
- **UI Components:** Minimum 3:1 ratio

### Focus States
- Visible focus indicators (blue ring)
- 2px outline minimum
- Never remove focus styles

### Alt Text
- Descriptive for images
- Empty for decorative images
- Concise and meaningful

### Keyboard Navigation
- All interactive elements accessible
- Logical tab order
- Skip links for navigation
